## 前端

- [「杭州 - 九月」Node Party 线下技术分享](https://cnodejs.org/topic/5b911f1837b3005a0b0e6c16)：NodeParty 品牌诞生于 CNode 社区，杭州 NodeParty 是由大搜车（Souche Inc.）发起的一个面向 Node.js 爱好者的技术分享聚会，希望能够构建一个持续的分享学习的平台，一起发展 Node.js 生态；
