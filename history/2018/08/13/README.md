## 新闻

- [印度计划限制美国科技巨头影响力](https://www.solidot.org/story?sid=57570)：根据新的电子商务法草案，印度用户在电商平台、社交媒体和搜索引擎产生的数据必须储存在印度境内，而且政府需要能获取这些数据。印度决策者和本国创业公司担心由于美国巨头的竞争它们难以发展壮大，他们看到了中国的限制政策如何帮助它创造出了世界知名的科技品牌，如阿里巴巴和腾讯；
